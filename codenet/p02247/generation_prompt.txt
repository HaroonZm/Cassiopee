Find places where a string
P
is found within a text
T
.
Print all indices of
T
where
P

T
start with 0.
Input
In the first line, a text
T

P
is given.
output
Print an index of
T
where
P
found in a line. Print the indices in ascending order.
Constraints
1 ≤ length of
T
≤ 1000
1 ≤ length of
P
≤ 1000
The input consists of alphabetical characters and digits
Sample Input 1
aabaaa
aa
Sample Output 1
0
3

xyz
Sample Output3

En te basant sur cet énoncé, réécris le code suivant tout en conservant exactement sa fonctionnalité.

```python
T = input()
P = input()

if len(P) > len(T):
    pass
else:
    p = len(P)
    ans = []
    for i in range(len(T)-len(P)+1):
        tmp = ""
        for j in range(i,i+p):
            tmp += T[j]
        if tmp == P:
            ans.append(i)
    
    for i in ans:
        print(i)
```

Counting Characters
Write a program which counts and reports the number of each alphabetical letter. Ignore the case of characters.
Input
A sentence in English is given in several lines.
Output
Prints the number of alphabetical letters in the following format:

.
.

Constraints
The number of characters in the sentence < 1200
Sample Input
This is a pen.

g : 0
h : 1

En te basant sur cet énoncé, réécris le code suivant tout en conservant exactement sa fonctionnalité.

```python
import sys

chklist = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j', 'k', 'l',
           'm', 'n', 'o', 'p', 'q', 'r', 's', 't', 'u', 'v', 'w', 'x', 'y', 'z']

word = ''
for i in sys.stdin:
    word += i

for n in chklist:
    cnt = 0
    for w in word:
        if n.upper() == w.upper():
            cnt += 1
    print(str.format('{} : {}', n, cnt))
```

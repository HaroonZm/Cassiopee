Voici un énoncé de problème :

Circle in a Rectangle
Write a program which reads a rectangle and a circle, and determines whether the circle is arranged inside the rectangle. As shown in the following figures, the upper right coordinate $(W, H)$ of the rectangle and the central coordinate $(x, y)$ and radius $r$ of the circle are given.
Input
Five integers $W$, $H$, $x$, $y$ and $r$ separated by a single space are given in a line.
Output
Print "Yes" if the circle is placed inside the rectangle, otherwise "No" in a line.
Constraints
$ -100 \leq x, y \leq 100$
$ 0 < W, H, r \leq 100$
Sample Input 1
5 4 2 2 1
Sample Output 1
Yes
Sample Input 2
5 4 2 4 1
Sample Output 2
No


En te basant sur cet énoncé, réécris le code suivant tout en conservant exactement sa fonctionnalité.

```python
w, h, x, y, r = eval(input().replace(' ', ','))
print("Yes" if 0 + r <= x <= w - r and 0 + r <= y <= h - r else "No")
```

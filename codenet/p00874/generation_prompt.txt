```python
while True:
    w,d = map(int,input().split())
    if w == 0 and d == 0:
        break

    h = list(map(int,input().split()))
    m = list(map(int,input().split()))

    hl = [0] * 30
    ml = [0] * 30

    for i in h:
        hl[i] += 1
    for i in m:
        ml[i] += 1

    ans = 0
    for i in range(30):

        ans += i * max(hl[i],ml[i])

    print (ans)
```

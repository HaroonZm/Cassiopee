Voici un énoncé de problème :

Print a Rectangle
Draw a rectangle which has a height of
H
cm and a width of
W
cm. Draw a 1-cm square by single '#'.
Input
The input consists of multiple datasets. Each dataset consists of two integers
H
and
W
separated by a single space.
The input ends with two 0 (when both
H
and
W
are zero).
Output
For each dataset, print the rectangle made of
H
×
W
'#'.
Print a blank line after each dataset.
Constraints
1 ≤
H
≤ 300
1 ≤
W
≤ 300
Sample Input
3 4
5 6
2 2
0 0
Sample Output
####
####
####
######
######
######
######
######
##
##


En te basant sur cet énoncé, réécris le code suivant tout en conservant exactement sa fonctionnalité.

```python
while True:
    H,W = [int(x) for x in input().split()]
    if H == 0 and W == 0:
        break
    for j in range(H):
        print("#"*W)
    print("")
```

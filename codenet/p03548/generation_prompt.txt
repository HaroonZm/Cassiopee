Problem Statement
We have a long seat of width
X
centimeters.
There are many people who wants to sit here. A person sitting on the seat will always occupy an interval of length
Y
centimeters.
We would like to seat as many people as possible, but they are all very shy, and there must be a gap of length at least
Z
centimeters between two people, and between the end of the seat and a person.
At most how many people can sit on the seat?
Constraints
All input values are integers.
1 \leq X, Y, Z \leq 10^5
Y+2Z \leq X
Input
Input is given from Standard Input in the following format:
X
Y
Z
Output
Print the answer.

Sample Output 1
3

En te basant sur cet énoncé, réécris le code suivant tout en conservant exactement sa fonctionnalité.

```python
x,y,z = map(int,input().split())
n = x // (z + y)
c = 0
for i in range(1,n+1):
    if ( y + z ) * i + z <= x:
        c = i
    else:
        break
print(c)
```

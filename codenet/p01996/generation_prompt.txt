Voici un énoncé de problème :

A: テスト
問題
$N$ 個の席が一直線上に並んでいる教室で $M$ 人の生徒がテストを受けることになった。
席には、前から $1 \dots N$ の番号が振られており、席 $1$ つにつき生徒 $1$ 人が座れる。
いま、 各生徒は、 $A_1, \dots, A_M$ 番の席に座っている。
テストを始めるためには、以下の条件を満たさなければならない。
$1 \dots M$ 番のどの席にも生徒が座っている。
そこで、条件を満たすまで次の操作を繰り返すことにした。
最も後ろに座っている生徒を移動させ、空いている席のうち最も前に座らせる。
条件を満たすために必要な操作回数を求めよ。
制約
入力値は全て整数である。
$1 \leq N \leq 1000$
$1 \leq M \leq N$
$1 \leq A_i \leq N$
$1 \leq i < j \leq M$ ならば $A_i < A_j$
入力形式
入力は以下の形式で与えられる。
$N\ M$
$A_1 \dots A_M$
出力
条件を満たすために必要な操作回数を出力せよ。また、末尾に改行も出力せよ。
サンプル
サンプル入力 1
6 4
1 4 5 6
サンプル出力 1
2
サンプル入力 2
10 3
1 2 3
サンプル出力 2
0


En te basant sur cet énoncé, réécris le code suivant tout en conservant exactement sa fonctionnalité.

```python
N, M = (int(x) for x in input().split())
A = (int(x) for x in input().split())

print(M - len([x for x in A if x <= M]))
```

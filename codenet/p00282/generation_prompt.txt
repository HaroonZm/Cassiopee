塵劫記
大きな数を表そうとすると、文字数も多くなるし、位取りがわからなくなってしまうので、なかなか面倒です。大きな数をわかりやすく表すために、人々は数の単位を使ってきました。江戸時代に書かれた「塵劫記」という本の中では、数の単位が次のように書かれています。
たとえば、２の１００乗のようなとても大きな数は、１２６穣７６５０(じょ)６００２垓２８２２京９４０１兆４９６７億３２０万５３７６と表せます。それでは、正の整数

```python
while 1:
  m,n=map(int,input().split())
  if m+n==0: break
  
  if m**n<10000:
    print(m**n)
    continue
  
  s=str(m**n)
  while len(s)%4>0: s="0"+s
  
  us=["","Man","Oku","Cho","Kei","Gai","Jo","Jou","Ko","Kan","Sei","Sai","Gok","Ggs","Asg","Nyt","Fks","Mts"]
  for i in range(0,len(s),4):
    x=int(s[i:i+4])
    if x==0: continue
    print(str(x)+us[(len(s)-i)//4-1],sep="",end="")
  print()
```
